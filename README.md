
### Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/A1-Opportunities-Africa.git
   cd A1-Opportunities-Africa
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Environment setup**
   ```bash
   cp .env.example .env.local
   # Configure your environment variables
   ```

4. **Start development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

5. **Open in browser**
   ```
   http://localhost:5173
   ```
